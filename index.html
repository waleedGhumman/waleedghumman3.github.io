Initial Rough Sketches

Since this was going to be our last project, we wanted it to be outstanding. Below are the initial sketches of our daily problems.

Sketch 1: Smart umbrella which blinks green if raining outside or red if no need.

Sketch 2: Smart key chain which blinks red if keys pressed against it or green if vacant.

Sketch 3: Smart lamp which turns off if no noise is detected like in TFDL.

Sketch 4: Posture-correct chair, it starts vibrating if human is not pressed against the chair correctly.

Sketch 5: Smart bed, which makes one part cold and the other hot.

Sketch 6: Smart water plant, blinks light if water needed or other problems.

Sketch 7: Smart medicine box, lights up the box based on the day medicines are needed.

Sketch 8: Smart trash bin, blinks red if full or green if space is available.

Sketch 9: Smart water bottle, blinks light when it's time to drink.

Sketch 10: Smart fridge which blinks the compartment of expiring food. You set a timer when food is about to expire.

Final Storyboard
