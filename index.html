Initial Rough Sketches

since this was going to be our last project, we wanted it to be outstanding. Below are the initial sketches of our daily problems

Sketch 1: smart umbrella which blinks green if raining outside or red if no need

Sketch 2: smart key chain which blinks red if keys pressed against it or green if vacant

Sketch 3: smart lamp which turns off if no noise is detected like in tfdl

Sketch 4: posture correct chair, it starts vibrating if human is not pressed against the chair correctly

Sketch 5: smart bed, which makes one part cold and other hot

Sketch 6: smart water plant, blinks light if water needed or other problems.

Sketch 7: smart medicine box, lights up the box based on the day that box medicines needed.

Sketch 8: smart trash bin, blinks red if full or green if space there

Sketch 9: smart water bottle, blinks light when time to drink

Sketch 10: smart fridge which blinks the compartment of expiring food, you essentially set a timer when you run out of that food or maybe is expiring

Final Storyboard

we chose Sketch 9 as final project because it popped on all teammates brain and it solved important problem of dehydration. But my version was basic and Kaylee came up with the tail idea.
